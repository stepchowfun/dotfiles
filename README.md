## Installation

Run this command:

```sh
sh -c "$(curl https://raw.githubusercontent.com/stepchowfun/dotfiles/main/install.sh -LSfs)"
```

**NOTE:** The installer script will overwrite existing dotfiles and other configuration. Use at your own risk!
