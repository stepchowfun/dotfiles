## Installation

1. Run this: `curl -sSf https://www.stephanboyer.com/dotfiles | sh`
2. Configure your terminal to use the `13pt Meslo LG S Regular for Powerline` font.

**NOTE:** The installer script uses [base16-shell](https://github.com/chriskempson/base16-shell) to set the terminal colors, and base16-shell is [not supported](https://github.com/chriskempson/base16-shell/issues/139) for macOS's Terminal app. To ensure the color scheme works correctly on macOS, use [iTerm2](https://www.iterm2.com/).
